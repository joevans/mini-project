# Mini Project

## Getting Started

### Installing

* Clone the repo:
   ```sh
   git clone https://github.com/joevans/mini-project.git
   ```
* Navigate to directory.
* Install the packages by running:
   ```sh
   npm install
   ```
* Run this to start the project:
   ```sh
   npm run build
   ```
* Open `index.html` in a browser.
